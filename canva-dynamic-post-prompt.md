## CANVA SPECIFIC INSTRUCTIONS

### In Canva Editor:

1. **Start Design:**
   - Select: "Instagram Post" template (1080x1350px)
   - OR: Custom size 1080x1350px

2. **Background:**
   - Option A: Use hero property photo as background (40% of height)
   - Option B: Create two-section design (image top, white bottom)
   - Add subtle gradient overlay on image section (dark, 30-40% opacity)

3. **Image Insertion (No Upload):**
   - Click "Upload an image" → "Link image"
   - Paste Vimeo or property URL
   - Canva will generate preview/thumbnail
   - Position and resize as needed

4. **Text Layers:**
   - Property name: Layer 1 (headline)
   - Price: Layer 2
   - Tagline: Layer 3
   - Date/time box: Layer 4 (shape + text)
   - Features grid: Layer 5 (text or table)
   - CTA: Layer 6
   - Footer: Layer 7

5. **Color Application:**
   - Apply gold (#c9a961) to:
     - Open House badge
     - Price display
     - Section dividers
     - CTA button
     - Feature bullet points
   - Apply dark (#1a1a1a) to:
     - Main headlines
     - Bold text
     - Borders

6. **Spacing & Alignment:**
   - Center-align most elements
   - Leave 40px padding on sides
   - Use consistent 16-20px spacing between sections
   - Keep design balanced (not cramped)

7. **Typography:**
   - Add font: Garamond (serif) for luxury feel
   - Add font: Inter (sans-serif) for modern touch
   - Use line-height: 1.6-1.8 for readability

8. **Shadows & Effects:**
   - Add text shadow to text over images (dark, 2-3px)
   - Add subtle drop shadow to CTA button
   - Add 1px border to feature boxes
   - Avoid heavy effects (keep luxury minimal)

9. **Final Design Checklist:**
   - [ ] All text is readable (good contrast)
   - [ ] Color scheme is consistent
   - [ ] Hero image is visually dominant
   - [ ] CTA button is prominent
   - [ ] No spelling errors
   - [ ] Date/time is clear
   - [ ] Property name is prominent
   - [ ] Design feels luxurious (minimal, elegant)

10. **Export:**
    - Download as JPG (high quality, 1080x1350px)
    - Save backup in Canva (for future edits)
    - Verify file size under 8MB
