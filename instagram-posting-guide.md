### Step 2: Optimize Image Size

- Format: JPG or PNG
- File size: Under 8MB
- Quality: High (no compression artifacts)

- Command line: `convert image.png -resize 1080x1350 instagram-post.jpg`

## PART 2: CREATE & POST THE INSTAGRAM REEL

### Step 1: Download CapCut
1. Mobile:
   - iOS: App Store → Search "CapCut" → Download
   - Android: Google Play → Search "CapCut" → Download

2. Desktop (Optional):
   - Go to: capcut.com
   - Download desktop version
   - Install

### Step 2: Gather Your Property Footage
Before opening CapCut, collect:
- [ ] Drone footage (aerial opening shot)
- [ ] Exterior footage (driveway, front)
- [ ] Pool/patio footage
- [ ] Interior kitchen footage
- [ ] Primary bedroom footage
- [ ] Primary bathroom footage
- [ ] Closing aerial shot

**Organize in phone camera roll:**
- Create album: "1076 Westover - Reel Footage"
- Or create folder on computer with same structure

### Step 3: Open CapCut & Create Project
1. Open CapCut app
2. Tap "Create a video" or "New project"
3. Select template (optional): Choose "Real Estate" or "Luxury" if available
4. Set to vertical format: 1080x1920 (should be default)

### Step 4: Import Footage in Order
**Follow the reel timeline (30 seconds total):**

| Order | Time | Footage | Duration |
|-------|------|---------|----------|
| 1 | 0-2s | Drone aerial opening | 2s |
| 2 | 2-4s | Drone wide view | 2s |
| 3 | 4-6s | Exterior front/approach | 2s |
| 4 | 6-8s | Pool/patio | 2s |
| 5 | 8-10s | Interior foyer/living | 2s |
| 6 | 10-12s | Kitchen showcase | 2s |
| 7 | 12-14s | Primary bedroom | 2s |
| 8 | 14-16s | Bathroom details | 2s |
| 9 | 16-18s | Pool close-up | 2s |
| 10 | 18-20s | Drone sunset/golden hour | 2s |
| 11 | 20-28s | Quick montage (rapid cuts) | 8s |
| 12 | 28-30s | Final aerial shot | 2s |

**How to import:**
1. Tap "+" (Add) at bottom
2. Select video or photo from gallery
3. Drag to timeline in correct order
4. Tap each clip → Trim to 2-3 seconds (use trim slider)
5. Repeat for all 12 clips

### Step 5: Add Text Overlays
**CapCut Text Feature:**

1. Tap "Text" in toolbar
2. Tap "Add text"
3. Type first text: "1076 Westover Rd"
4. Customize:
   - Font: Bold sans-serif
   - Size: Large (48+)
   - Color: Gold (#c9a961) or white
   - Position: Center, top
   - Duration: 2-4 seconds

**Repeat for:**
- "1076 Westover Rd" (0-2s)
- "Resort Living" (6-8s)
- "Chef's Kitchen" (10-12s)
- "Spa Primary Suite" (12-14s)
- "$2,399,000" (18-20s)
- Property details during montage (20-28s):
  - "4 Beds"
  - "4.5 Baths"
  - "4,531 SF"
  - "1.04 Acres"
- Final screen (28-30s):
  ```
  OPEN HOUSE
  SUNDAY 1-4 PM
  Link in Bio
  ```

### Step 6: Add Music
1. Tap "Music" in toolbar
2. Search for upbeat/luxury music
3. Suggestions:
   - "Elegant"
   - "Luxury"
   - "Modern"
   - "Upbeat Real Estate"
4. Select track
5. Trim music to fit 30 seconds
6. Adjust volume (usually 70-80%)

### Step 7: Add Transitions (Optional)
1. Click between clips
2. Tap "Transition"
3. Select:
   - "Fade" (between locations)
   - "Cut" (quick changes)
   - "Zoom" (close-ups)
4. Duration: 0.3-0.5 seconds

### Step 8: Add Effects (Optional)
1. Tap "Effects"
2. Add subtle effects:
   - Color correction (warm/golden tone)
   - Vignette (slight edge darkening)
   - Slight zoom/motion on still shots
3. Keep minimal - let footage speak

### Step 9: Color Grade
1. Tap "Tools" → "Color"
2. Adjust:
   - Brightness: +10-15%
   - Saturation: +5-10% (enhance luxury feel)
   - Contrast: +10%
3. Make it warm/golden tone

### Step 10: Preview
1. Tap play button to preview full reel
2. Check timing
3. Verify text overlays are readable
4. Make sure transitions flow smoothly
5. Verify music fits the mood

### Step 11: Export
1. Tap "Export" or checkmark
2. Select resolution: 1080p or 4K
3. Select aspect ratio: 9:16 (vertical for Instagram)
4. Tap "Export"
5. Choose quality: High
6. Wait for processing (2-10 minutes depending on device)

### Step 12: Save to Device
1. Once exported, tap "Save" or "Share"
2. Choose "Save video" (for later posting)
3. Video will be saved to camera roll

## TROUBLESHOOTING

**"Video too long"** → Trim to under 60 seconds (Reels max)
**"Caption too long"** → Move hashtags to first comment
**"Upload failed"** → Check internet, try again in WiFi
**"Text unreadable"** → Increase font size, add text shadow
**"Video looks dark"** → Add brightness in CapCut color grading
